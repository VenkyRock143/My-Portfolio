## Featured Projects

### 🌤 SkyCast — AI Weather Dashboard
A production-grade multi-user weather platform that tracks unlimited global cities with live data. I integrated **Groq's LLaMA 3.3 70B via LangChain** to power a floating AI assistant with persistent conversation context, auto-generated city briefings, and natural-language weather queries.

Built with: `Next.js 15` `React 19` `TypeScript` `Node.js` `Express` `MongoDB` `Groq API` `LangChain`
→ JWT auth, NodeCache TTL, Express rate limiting, and a full CI/CD pipeline via GitHub Actions

### 🎰 Crypto Crash — Real-Time Multiplayer Game
A real-time multiplayer crash game built on Node.js and Socket.IO. The server handles concurrent WebSocket events, game logic validation, and live crash-multiplier broadcasting to all connected clients simultaneously.

Built with: `React` `Socket.IO` `Node.js` `Express` `MongoDB` `CoinMarketCap API`
→ MVC architecture, modular REST API, complete setup documentation
